# 親水披膜（Hydrophilic Coating）

## 主題定位
- **目標**：建立可量產之親水性低摩擦披膜——材料、前處理、塗佈、測試、生物安全風險與量產導入
- **對應產品**：Sheath、微導管、血攝；基材 **ETFE、Pebax**
- **負責**：邱長儒、林榮峰
- **策略地圖階段**（依五年路線圖）：塗層材料認知／表面前處理 → 塗佈製程 → 性能測試 → 失效分析 → 生物安全風險 → 量產導入

## 技術重點
- 塗層材料認知：親水高分子種類、與 ETFE/Pebax 的附著策略（→ 詳見 [塗層材料與化學](B1-塗層材料與化學.md)）
- 表面前處理：ETFE 表面能低，前處理是成敗關鍵（→ [表面處理與塗佈製程](B2-表面處理與塗佈製程.md)）
- 塗佈製程：浸塗／噴塗／線上塗的取捨
- 性能測試：摩擦係數、潤滑耐久（cycle 數）、乾濕態差異（→ [性能測試與失效分析](B3-性能測試與失效分析.md)）
- 失效模式：塗層脫落、潤滑衰減、微粒（particulate）風險
- 生物安全：溶出物、微粒、ISO 10993 相關（→ [生物安全與法規](B4-生物安全與法規.md)）
